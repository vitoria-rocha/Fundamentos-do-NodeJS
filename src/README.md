Nao deve ser possivel:

- [ x ] Cadastrar uma conta com CPF ja existente

- [ ] Fazer saque em uma conta nao existente
- [ ] Excluir uma conta nao existente
- [ ] Fazer saque quando o aldo for insuficiente
